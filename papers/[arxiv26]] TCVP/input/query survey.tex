  title=\textbf{Annotator Instructions: Realistic Query Selection},
  fonttitle=\scriptsize\bfseries\sffamily
]
\scriptsize
\setlength{\parindent}{0pt}
\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0.25ex, topsep=0.25ex, parsep=0pt, partopsep=0pt]
  \item You will be given a YouTube video, a specific moment from that video, and a list of possible text queries.
  \item Imagine the following situation.
  \begin{enumerate}[label=\alph*., leftmargin=1.8em, itemsep=0.2ex, topsep=0.2ex, parsep=0pt, partopsep=0pt]
    \item You previously watched the entire YouTube video.
    \item Later, you want to find that specific moment again using a text query.
    \item You have an AI system that can locate the exact timestamp if you provide a text description of the moment.
  \end{enumerate}
  \item From the list below, choose the query you would most likely give to the AI system.
  \item Select a natural and realistic search query.
  \item \texttt{Natural} means it sounds like something a real person would type.
  \item \texttt{Realistic} means it is clear and not overly long.
\end{enumerate}
\end{tcolorbox}
