정보이론 기초: 엔트로피, 교차 엔트로피, KL Divergence

불확실성을 수로 표현하는 방법 — 섀넌 엔트로피부터 KL divergence까지, 확률 분포 간의 차이를 측정하는 도구들이 어떻게 딥러닝의 핵심 손실 함수와 생성 모델의 목적 함수로 이어지는지 정리한다.

Math · Mar 4, 2026 · 20 min read

정보를 측정한다는 것

"정보"를 직관적으로 생각해보자. 오늘 서울에 비가 내렸다는 사실은 그리 놀랍지 않다. 하지만 6월 한낮에 눈이 내렸다는 소식은 대단히 놀랍다. 이 놀라움의 차이가 바로 정보량의 차이다.

정보이론의 핵심 직관은 이렇다: 희귀한 사건일수록 더 많은 정보를 담는다. 확실히 일어날 사건(확률 1)은 아무 정보도 주지 않는다. 이미 아는 것이기 때문이다. 반면 거의 불가능한 사건이 일어났을 때, 우리는 큰 정보를 얻는다.

1948년 클로드 섀넌(Claude Shannon)은 이 직관을 수식으로 엄밀하게 만들었다. 그 결과물이 오늘날 ML과 딥러닝 전반을 관통하는 정보이론의 기초다.

자기 정보량 (Self-Information)

사건 $x$의 정보량은 그 확률 $P(x)$의 역수에 로그를 취한 값으로 정의한다.

자기 정보량 (Self-Information)

$$I(x) = -\log P(x) = \log \frac{1}{P(x)}$$

로그의 밑이 2이면 단위는 비트(bit), 자연 로그($e$)이면 나트(nat)다. ML에서는 주로 나트를 쓴다.

성질을 확인해보자:

  • $P(x) = 1$이면 $I(x) = 0$. 확실한 사건은 정보가 없다.
  • $P(x) \to 0$이면 $I(x) \to \infty$. 불가능에 가까운 사건이 일어났을 때 정보가 무한히 크다.
  • 두 독립 사건 $x, y$의 결합 정보량: $I(x, y) = -\log P(x)P(y) = I(x) + I(y)$. 정보는 더해진다.
비트 해석 예시: 공정한 동전 1번 → $I(H) = -\log_2(0.5) = 1$ bit. 공정한 주사위의 특정 면 → $I = \log_2 6 \approx 2.58$ bits. 100면체 주사위의 특정 면 → $I = \log_2 100 \approx 6.64$ bits. 이진 부호화에서 이 정보를 저장하는 데 필요한 최소 비트 수다.

$I(x) = -\log_2 P(x)$. $P(x) \to 0$일수록 정보량은 무한대로 증가하고, $P(x) = 1$이면 정보량은 0이다.

섀넌 엔트로피 (Shannon Entropy)

자기 정보량은 특정 사건 하나의 정보량이다. 그런데 확률 변수 $X$ 전체의 "평균적인 불확실성"은 어떻게 측정할까? 자기 정보량의 기댓값이 바로 섀넌 엔트로피다.

섀넌 엔트로피 (Shannon Entropy)

$$H(X) = \mathbb{E}[I(X)] = -\sum_{x} P(x) \log P(x)$$

연속 확률 변수의 경우(미분 엔트로피):

$$h(X) = -\int p(x) \log p(x)\, dx$$

엔트로피의 성질

  • 비음성: $H(X) \geq 0$. $0 \log 0 = 0$으로 정의하면(극한 처리) 항상 성립.
  • 결정론적 분포에서 최소: $P(x^*) = 1$이면 $H = 0$. 결과가 완전히 확정되면 불확실성이 없다.
  • 균등 분포에서 최대: $n$개의 이산 값에서 균등 분포일 때 $H = \log n$이 최대다 (Jensen 부등식으로 증명 가능). 모든 결과가 동등하게 불확실할 때 불확실성이 가장 크다.
  • 가산성: $X$, $Y$가 독립이면 $H(X, Y) = H(X) + H(Y)$.
이진 엔트로피 예시: 베르누이 확률 변수 $X \sim \text{Bernoulli}(p)$의 엔트로피: $H = -p\log p - (1-p)\log(1-p)$. $p = 0.5$일 때 $H = 1$ bit로 최대. $p = 0$ 또는 $p = 1$일 때 $H = 0$. 동전이 공정할수록 불확실성이 크다.

이진 엔트로피 $H(p) = -p\log_2 p - (1-p)\log_2(1-p)$. $p=0.5$에서 최대(1 bit), $p=0$ 또는 $1$에서 0.

엔트로피는 분포의 불확실성을 수치화한다. 동시에 그 분포에서 오는 메시지를 최적으로 부호화했을 때 필요한 평균 비트 수이기도 하다.

교차 엔트로피 (Cross-Entropy)

현실에서는 실제 분포 $p$를 모르고, 우리의 모델이 근사 분포 $q$를 제시한다. $p$에서 나온 메시지를 $q$에 최적화된 부호로 인코딩하면 얼마나 많은 비트가 필요할까? 이것이 교차 엔트로피다.

교차 엔트로피 (Cross-Entropy)

$$H(p, q) = -\sum_x p(x) \log q(x) = \mathbb{E}_{x \sim p}[-\log q(x)]$$

$p$: 실제 분포(데이터 생성 분포)  ·  $q$: 모델이 추정한 근사 분포

$H(p, q) \geq H(p)$ 임이 항상 성립한다. 잘못된 코드($q$)를 쓰면 항상 최적($p$ 기준) 보다 더 많은 비트를 써야 한다. 등호는 $p = q$일 때만 성립한다.

분류 손실 함수로서의 교차 엔트로피

분류 문제에서 실제 레이블 $y$가 클래스 $c$에 해당한다면, 실제 분포는 one-hot이다: $p(c) = 1[y = c]$. 모델의 출력은 softmax 확률 $q(c) = \hat{p}_c$라 하면:

분류에서의 교차 엔트로피 손실
$$H(p, q) = -\sum_{c=1}^C \mathbf{1}[y = c] \log \hat{p}_c = -\log \hat{p}_y$$

정답 클래스의 예측 확률에 로그를 취하고 부호를 바꾼 값. MLE의 NLL과 동일하다.

교차 엔트로피를 최소화하는 것은 KL divergence를 최소화하는 것과 같다. 왜냐하면 $H(p, q) = H(p) + D_{KL}(p \| q)$이고, $H(p)$는 모델 파라미터에 무관한 상수이기 때문이다.

$H(p,q) = H(p) + D_{KL}(p \| q)$. 교차 엔트로피 최소화 = KL 최소화. $H(p)$는 상수이므로.

KL Divergence

두 분포 $p$와 $q$가 얼마나 다른지 측정하는 함수가 필요하다. KL divergence(Kullback–Leibler divergence, 상대 엔트로피)는 그 측도 중 가장 중요한 것이다.

KL Divergence (Relative Entropy)

$$D_{KL}(p \| q) = \sum_x p(x) \log \frac{p(x)}{q(x)} = \mathbb{E}_{x \sim p}\left[\log \frac{p(x)}{q(x)}\right]$$

교차 엔트로피와의 관계:

$$D_{KL}(p \| q) = H(p, q) - H(p)$$

"$p$를 기준으로 $q$가 얼마나 다른가"를 측정한다. $q$를 $p$에 맞추는 데 드는 추가 비용이다.

비음성 증명: Jensen 부등식

$D_{KL}(p \| q) \geq 0$은 Jensen 부등식으로 우아하게 증명된다. $\log$는 오목 함수(concave)이므로:

Gibbs' Inequality — KL ≥ 0
$$-D_{KL}(p \| q) = \mathbb{E}_{p}\left[\log\frac{q(x)}{p(x)}\right] \leq \log \mathbb{E}_{p}\left[\frac{q(x)}{p(x)}\right] = \log \sum_x p(x) \cdot \frac{q(x)}{p(x)} = \log \sum_x q(x) = \log 1 = 0$$ $$\therefore\quad D_{KL}(p \| q) \geq 0, \quad \text{등호 } \Leftrightarrow p = q$$

비대칭성

KL divergence는 비대칭이다: 일반적으로 $D_{KL}(p \| q) \neq D_{KL}(q \| p)$. 이 때문에 KL divergence는 거리(metric)가 아니다. 삼각 부등식도 성립하지 않는다.

그런데 이 비대칭성은 단점이 아니라 의미 있는 구분이다. 어느 방향의 KL을 최소화하느냐에 따라 최적화의 행동이 완전히 달라진다.

Forward KL vs Reverse KL

$p$를 실제 분포, $q_\phi$를 우리가 학습할 근사 분포라 하자.

같은 이봉 분포 $p$를 단봉 $q$로 근사할 때: Forward KL은 두 모드를 모두 커버하는 넓은 $q$를, Reverse KL은 하나의 모드에만 집중하는 좁은 $q$를 만든다.

1
Forward KL: $D_{KL}(p \| q_\phi)$ — Mode-Covering ▼
$p(x) > 0$인 곳에서 $q_\phi(x) = 0$이 되면 $\log(p/q) \to \infty$가 되어 무한 패널티가 발생한다. 따라서 $q_\phi$는 $p$의 모든 지지(support)를 반드시 커버해야 한다. 다봉 분포(multi-modal) $p$가 있으면 $q_\phi$는 모든 모드(mode)를 포함하는 넓은 분포가 된다. MLE는 $\min_\phi D_{KL}(p_{data} \| q_\phi)$와 동치이므로, 표준 학습이 여기에 해당한다.
2
Reverse KL: $D_{KL}(q_\phi \| p)$ — Mode-Seeking ▼
$q_\phi(x) > 0$인 곳에서 $p(x) = 0$이 되면 무한 패널티가 발생한다. 따라서 $q_\phi$는 $p$가 낮은 곳에 확률을 두지 않는다. 대신 $p$의 단 하나의 강한 모드에만 집중하는(zero-forcing, 확률을 0으로 몰아가는) 행동을 한다. Variational Inference(VI)는 $\min_\phi D_{KL}(q_\phi \| p)$를 목적으로 하며, 다봉 사후 분포를 단봉으로 근사하는 경향이 있다.
3
JS Divergence — 대칭 버전 ▼
Jensen–Shannon divergence: $D_{JS}(p \| q) = \frac{1}{2}D_{KL}(p \| m) + \frac{1}{2}D_{KL}(q \| m)$, 여기서 $m = \frac{p+q}{2}$. 대칭적($D_{JS}(p\|q) = D_{JS}(q\|p)$)이고 유계($0 \leq D_{JS} \leq \log 2$)이다. 원래 GAN의 학습 목적 함수가 generator-discriminator 균형점에서 JS divergence 최소화와 동치임이 알려져 있다.

상호 정보량 (Mutual Information)

두 확률 변수 $X$와 $Y$가 얼마나 서로 연관되어 있는가를 측정하는 양이다. 결합 분포 $P(X, Y)$와 독립 가정 하의 주변 분포의 곱 $P(X)P(Y)$ 사이의 KL divergence로 정의한다.

상호 정보량 (Mutual Information)

$$I(X;\, Y) = D_{KL}(P(X,Y) \| P(X)P(Y)) = \sum_{x,y} P(x,y)\log\frac{P(x,y)}{P(x)P(y)}$$

엔트로피로 표현하면:

$$I(X;\, Y) = H(X) - H(X \mid Y) = H(Y) - H(Y \mid X) = H(X) + H(Y) - H(X, Y)$$

$X$에 대한 불확실성이 $Y$를 알고 나서 얼마나 줄어드는가. 항상 $\geq 0$.

$I(X;Y) = 0$은 $X$와 $Y$가 독립임을 의미한다. 특성 선택(feature selection), 정보 병목 이론(Information Bottleneck), 그리고 표현 학습에서 "어떤 정보를 보존할 것인가"를 정량화할 때 핵심 역할을 한다.

VAE와 ELBO: KL divergence의 실제 응용

VAE(Variational Autoencoder)는 정보이론 개념이 생성 모델에 어떻게 녹아드는지를 보여주는 가장 명쾌한 예다. 잠재 변수 $z$를 통해 데이터 $x$를 생성하는 모델을 생각하자.

우리가 원하는 것은 $\log p_\theta(x)$를 최대화하는 것이다. 하지만 사후 분포 $p_\theta(z \mid x)$는 계산이 불가능(intractable)하다. 대신 근사 사후 분포 $q_\phi(z \mid x)$를 도입하면:

ELBO 유도
$$\log p_\theta(x) = \mathbb{E}_{q_\phi(z \mid x)}\left[\log p_\theta(x)\right]$$ $$= \mathbb{E}_{q_\phi}\left[\log \frac{p_\theta(x, z)}{p_\theta(z \mid x)}\right]$$ $$= \mathbb{E}_{q_\phi}\left[\log \frac{p_\theta(x, z)}{q_\phi(z \mid x)}\right] + D_{KL}(q_\phi(z \mid x) \| p_\theta(z \mid x))$$ $$\geq \underbrace{\mathbb{E}_{q_\phi}\left[\log p_\theta(x \mid z)\right] - D_{KL}(q_\phi(z \mid x) \| p(z))}_{\mathcal{L}_{\text{ELBO}}}$$

$D_{KL} \geq 0$이므로 $\log p_\theta(x) \geq \mathcal{L}_{\text{ELBO}}$, 즉 ELBO는 하한이다. ELBO를 최대화하면 두 가지 효과가 동시에 발생한다:

1
재구성 항: $\mathbb{E}_{q_\phi}[\log p_\theta(x \mid z)]$ ▼
잠재 변수 $z$를 샘플링했을 때 원래 데이터 $x$를 잘 복원하는가를 측정한다. $p_\theta(x \mid z)$가 가우시안이면 이 항은 MSE에 해당한다. Bernoulli이면 binary cross-entropy다. 디코더(decoder)의 재구성 품질을 높인다.
2
KL 정규화 항: $-D_{KL}(q_\phi(z \mid x) \| p(z))$ ▼
인코더가 만드는 사후 분포 $q_\phi(z \mid x)$를 사전 분포 $p(z) = \mathcal{N}(0, I)$에 가깝게 유지한다. 잠재 공간이 정규 분포를 따르도록 강제해 새로운 샘플 생성이 가능하게 한다. $q_\phi$가 가우시안 $\mathcal{N}(\mu, \sigma^2 I)$이면 이 항은 닫힌 형태로 계산 가능하다: $D_{KL} = \frac{1}{2}\sum_j (\mu_j^2 + \sigma_j^2 - \log \sigma_j^2 - 1)$.
ELBO의 의미: VAE는 "데이터를 잘 설명하는 잠재 표현(재구성 항)"과 "잠재 공간이 사전 분포에서 크게 벗어나지 않아야 한다는 정규화(KL 항)" 사이의 균형을 학습한다. 이 균형이 생성 모델의 핵심이다.

모든 개념의 연결 구조

개념 수식 ML에서의 역할
자기 정보량 $-\log P(x)$ 단일 샘플의 NLL (손실 기여도)
엔트로피 $H(p)$ $-\mathbb{E}_p[\log p]$ 불확실성 측도, 최적 부호화 비트 수
교차 엔트로피 $H(p,q)$ $-\mathbb{E}_p[\log q]$ 분류 손실 함수, MLE와 동치
KL divergence $D_{KL}(p\|q)$ $H(p,q) - H(p)$ VAE 정규화 항, VI 목적 함수
상호 정보량 $I(X;Y)$ $H(X) - H(X|Y)$ 특성 선택, 정보 병목, 표현 학습

마치며

섀넌이 1948년에 세운 정보이론은 원래 통신 채널의 용량을 분석하기 위한 것이었다. 그런데 70여 년이 지난 지금, 이 이론은 딥러닝의 거의 모든 곳에 살아있다.

교차 엔트로피 손실을 쓸 때마다, KL divergence로 사후 분포를 정규화할 때마다, 특성 간 상호 정보량을 최대화하는 SSL(self-supervised learning) 목적 함수를 볼 때마다 — 섀넌의 아이디어가 코드 안에서 작동하고 있는 것이다.

앞 포스트에서 다룬 MLE와 MAP도 이 맥락에서 다시 읽힌다. MLE는 $D_{KL}(p_{data} \| q_\theta)$를 최소화하는 것이고, 교차 엔트로피 손실은 그 practical한 구현이다. 수식이 아니라 개념의 흐름이 보이기 시작하면, 논문의 목적 함수가 낯설지 않아진다.
